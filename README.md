# java-Vehicle-Rental-System
A java console application using OOP and File I/O
A terminal-based application designed to manage Fleet of rental vehicle.

^Features^
- Add vehicles -- Supports Cars, Bikes and Vans with specific Attributes.
- Persistent Storage -- Saves and Loads data from a text file.(vehicle.txt)
- Dynamic Pricing -- Uses Polymorphism to calculate rental costs based on vehicle type.
- Search & manage -- Search by ID, rent vehicle and track total income.

  Tech Concepts that used
  - Inhertance -- Shared logic in a 'Vehicle' parent class.
  - Encapsulation -- Private fields with public getters and stters.
  - File I/O -- Reading/Writter data for persistence.
